Realtime Traffic Offence Detection to Increase Road Safety

Introduction

This project successfully developed a computer vision-based system for detecting traffic offenses in real time. The system identifies various violations, including exceeding speed limits, unauthorized vehicles, running traffic signals, illegal parking, driving in the wrong direction, and motorbike riders without helmets. It comprises four main modules: vehicle detection, vehicle classification, vehicle tracking, and traffic offense detection.

The vehicle detection module identifies vehicles on the road using visual data from live camera feeds. After detection, the vehicles are categorized into different types using the vehicle classification module. The vehicle tracking module monitors the movement of vehicles through traffic, while the traffic offense detection module analyzes traffic patterns to detect various types of violations as they occur.

The system was implemented using the OpenCV Deep Neural Network (DNN) module. YOLOv4 was used for accurate vehicle detection, and YOLOv4-tiny was employed for detecting motorbike riders without helmets. The DeepSORT algorithm enabled real-time vehicle tracking. The completed system achieved 86% accuracy for vehicle detection with YOLOv4 and 92% accuracy for helmet detection with YOLOv4-tiny.
